# Python Scripts

Main control scripts for the system in REAL Qcar .
That send all the core script to the Qcar by Wifi

## Key Scripts

- **start_refactored.py** - Launch refactored multi-vehicle system via SSH
- **stop_refactored.py** - Stop all running vehicles
- **multi_probing.py** / **one_probing.py** - To recived camera image from Qcar to Ground Station PC
- **hardware_stop.py** - Emergency hardware shutdown

- **calibrate.py** - LiDAR calibration utility
- **start.py** / **stop.py** - Legacy vehicle control scripts (Example of Quanser)
